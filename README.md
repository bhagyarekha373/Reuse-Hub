## Project Info

**Repository URL:** [https://github.com/BOGGARAPUSHAMITHA/Smarter-reuse-hub-through-Ai-Innovation](https://github.com/BOGGARAPUSHAMITHA/Smarter-reuse-hub-through-Ai-Innovation)

**Description:**  
This project aims to create a smarter reuse hub, allowing users to interact with items, buy, sell, and manage listings through an AI-assisted interface.

---

## Features

- Browse, search, and filter reusable items
- Add, edit, and manage items
- AI-powered chatbot integration
- User authentication and profile management
- Order management and checkout
- Responsive design with modern UI components

---

## Technologies Used

- **Vite** – Fast frontend build tool
- **React** – Frontend library
- **TypeScript** – Type-safe JavaScript
- **Tailwind CSS** – Utility-first CSS framework
- **shadcn-ui** – UI component library
- **Supabase** – Backend, authentication, and database
